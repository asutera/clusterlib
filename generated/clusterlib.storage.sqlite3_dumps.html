    clusterlib.storage.sqlite3_dumps¶

    clusterlib.storage.sqlite3_dumps(fname, key, value, timeout=7200.0)¶

    Dumps value with key in the sqlite3 database

    Parameters:

    fname : str

    path to the sqlite database

    key : str

    Key to the object. If the key is already present in the database, it will raise an exception

    value : object

    Object to stored

    timeout : float, (default=7200.0)

    The timeout parameter specifies how long the connection should wait for the lock to go away until raising an exception.
